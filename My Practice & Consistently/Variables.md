What is a Variable?

A variable is a container that stores data (a value). We can change the stored value anytime during program execution.

Variable Naming Rules

- Variable names can start with letters ("a-z", "A-Z") or an underscore ("_").
- They cannot start with a number.
- Numbers are allowed after the first letter (e.g., "student1").
- Only the underscore ("_") is allowed as a special character.
- Variable names are case-sensitive ("name" and "Name" are different).
